Informativa sulla Privacy e Trattamento Dati (Google OAuth)

Data di ultima revisione: 6 luglio 2026

Il Dott. Parrinello Giovanni (di seguito "il Professionista"), in qualità di Titolare del Trattamento, si impegna a proteggere la privacy degli utenti che utilizzano la piattaforma e il software "Nuova Energia" (accessibile tramite il sito web https://www.nuovaenergianutri.it).

1. Dati raccolti tramite le API di Google (OAuth 2.0)

Il software sviluppato da Nuova Energia richiede l'autorizzazione dell'utente per accedere ai seguenti servizi Google tramite protocollo sicuro OAuth 2.0:

  • Google Calendar: per la sincronizzazione, visualizzazione e gestione degli appuntamenti dello studio.
  • Gmail: per la visualizzazione delle email ricevute dai pazienti e per l'invio automatizzato di promemoria e comunicazioni relative alle visite.
  • Google Forms (Moduli Google): per l'importazione dei dati anamnestici compilati dai pazienti.

2. Finalità e conformità all'uso dei dati (Google Limited Use Policy)

L'uso delle informazioni ricevute dalle API di Google da parte del software rispetta rigorosamente le Norme sui dati dei servizi API di Google, inclusi i requisiti per l'Uso Limitato (Limited Use Requirements).

I dati vengono utilizzati esclusivamente per fornire le funzionalità operative descritte e non vengono memorizzati su server esterni al dispositivo dell'utente.

Nello specifico:

  • Non condividiamo, vendiamo o trasmettiamo in nessun caso i dati personali a terze parti.
  • Non utilizziamo i dati ottenuti tramite le API di Google per scopi pubblicitari, profilazione commerciale o per il tracciamento del comportamento dell'utente.
  • L'autenticazione avviene sul cloud sicuro di Google; il software non entra mai in possesso della password dell'account.

3. Responsabilità sui Moduli Google (Google Forms)

L'importazione dei dati avviene localmente sul dispositivo dell'utente. La cancellazione e la successiva gestione dei moduli sorgente su Google Forms rimangono sotto l'esclusiva responsabilità del professionista utilizzatore, in qualità di Titolare autonomo del Trattamento.

4. Diritti dell'utente e revoca del consenso

L'utente può revocare in qualsiasi momento l'accesso del software ai propri dati Google direttamente dalle impostazioni di sicurezza del proprio Account Google (sezione "App di terze parti con accesso al conto"), oppure interrompendo l'uso delle funzioni integrate.
